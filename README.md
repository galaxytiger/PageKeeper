# PageKeeper

PageKeeper is a JavaFX desktop application that helps users organize and track their reading. Users can create an account, log in, add books to a personal library, monitor reading progress, and save ratings or short reviews.

## Features

- User login and registration
- Personal book library dashboard
- Add new books with title, author, and reading status
- Track current page and total pages
- Mark books as read
- View detailed book information
- Save ratings and short reviews
- Track borrowed books and return dates

## Application Overview

PageKeeper is designed for users who want a simple way to manage their reading activity in one place. The app supports multiple reading states such as:

- Want to Read
- In Progress
- Read

Users can quickly view their collection, update progress, and keep notes on completed books.

## Main Screens

### Login / Registration
Users can log in with an existing account or create a new account by selecting a role, username, and password.

### Library Dashboard
The main dashboard displays the user’s books in a table with columns for:

- Title
- Status
- Progress
- Actions

Users can refresh the list, update progress, or mark a book as read.

### Add Book
Users can add a new book by entering:

- Title
- Author
- Status
- Current page
- Total pages
- Borrowed status
- Return date

### Book Details
The details view shows:

- Title
- Author
- Reading status
- Progress
- Rating
- Review

Users can update progress, mark the book as read, and save a review.
